# 付録C. 論文投稿前チェックリスト

各章で学んだ「論文公開を見据えた作法」を、投稿前に一覧で総点検するためのチェックリストである。
各項目から該当章への相互参照リンクを付けているので、詳細はそちらを参照する。

## 1. コード

- [ ] GitHubリポジトリが公開設定になっている
- [ ] ライセンスファイル（LICENSE）がある — [§7-2 GitHubの活用](./07_git.md#7-2-githubの活用)
- [ ] CITATION.cff がある（GitHubの「Cite this repository」ボタンが機能する） — [§7-2 GitHubの活用](./07_git.md#7-2-githubの活用)
- [ ] GitHub Releasesでバージョンがタグ付けされている — [§7-2 GitHubの活用](./07_git.md#7-2-githubの活用)
- [ ] Zenodo連携でDOIが発行されている — [§7-2 GitHubの活用](./07_git.md#7-2-githubの活用)
- [ ] READMEに目的・依存関係・実行手順・データ入手方法・引用方法が書かれている — [§18-2 プロジェクトドキュメント](./18_documentation.md#18-2-プロジェクトドキュメント)

## 2. データ

- [ ] embargo期間が適切に設定されている — [§19-3 データのダウンロードとローカル管理](./19_database_api.md#19-3-データのダウンロードとローカル管理)
