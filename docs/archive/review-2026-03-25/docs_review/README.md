# 原稿レビュー台帳

本ディレクトリは『AIエージェントと学ぶ バイオインフォマティクスプログラミングの作法』の、完了済みレビュー成果物アーカイブである。

未達成レビューと未完了修正の現行正本は `docs/review/` に戻してあり、本ディレクトリには完了済みの計画書・要約・進捗台帳・チェック結果を保存している。

## 生成物

- `chapter_review_sheet.csv`: 章ごとのレビュー進捗、観点別担当、機械監査の件数
- `section_inventory.csv`: 見出しとアンカーの棚卸し
- `initial_review_report.md`: 初回自動監査の要約
- `code_check.md`: 現在の pytest 実行結果、警告、未カバースクリプトの要約
- `structure_check.json`: 構造規約チェックの現在結果
- `xref_check.json`: 相互参照チェックの現在結果

## 現行正本

- 未達成レビュー一覧: `/docs/review/pending_review_items.md`
- 未完了修正一覧: `/docs/review/master_issue_log.csv`
- 参照台帳: `/docs/review/reference_registry.csv`
- URL チェック現行結果: `/docs/review/url_check.json`
