# AIエージェントを使いこなす はじめてのバイオインフォマティクス開発作法

-- 情報技術の基礎から環境構築・設計・テスト・公開まで

## 概要

AIコーディングエージェント（Claude Code CLI / Codex CLI）との協働を前提として、
バイオインフォマティクスのプログラミングに必要な知識を体系的に解説する書籍。

* AI for Science 時代に必要な研究開発の土台を、バイオインフォマティクスを通して学ぶ
* AIエージェントと進める研究開発で、雰囲気に流されずに手を動かすための一冊
* 配列解析と機械学習を題材に、研究ソフトウェア開発の基礎体力を身につける

各章の文字数・読了時間は [docs/chapter_stats.md](docs/chapter_stats.md) を参照（本文合計 約30万文字、読了 約30時間）。

注:引用に誤りが残っている可能性があります。ご指摘頂けると助かります。
